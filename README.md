# Screenshot Layer

  This Screenshot Layer module enable user to capture the customiziable snapshots of any websites
  
  Dependencies:
  
  * Mendix Modeler - 9.18.4 or latest
  * API Key - https://screenshotlayer.com/dashboard
  
  Features and Limitations:
  
  This module allows the user to convert and download the any kind of URL to mutiple Image format screenshots
  
  Configuration:
  
  * Create a account on ScreenshotLayer - https://screenshotlayer.com/
  * Get the API key from the dashboard - https://screenshotlayer.com/dashboard
  * From Mendix, add the ScreenShot_Overview page
  * Paste any of the URL in the URL text box and select any of the file format which needs to get converted
  * Click the convert button to fetch the snapshot of the respective URL page in the selected format
